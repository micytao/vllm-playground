## üõ†Ô∏è MCP Quick Start

### Installation

> ‚ö†Ô∏è **MCP requires Python 3.10 or higher**

MCP support requires the `mcp` package:

```bash
# Install vLLM Playground with MCP support
pip install vllm-playground[mcp]

# Or add MCP to existing installation
pip install mcp
```

### Usage

1. **Navigate to MCP Servers** in the sidebar
2. **Add a server** using a preset (e.g., Filesystem)
3. **Configure** the required variables (e.g., directory path)
4. **Connect** to the server
5. **Enable MCP** in the chat toolbar (üîó icon)
6. **Chat** and approve tool calls as they come

```
User: List the files in /home/user/documents
Assistant: [MCP Tool Call: list_directory]
‚Üí Click "Execute" to approve
```

> **Note:** Enable "Tool Calling" in Server Configuration and set Max Model Length to 8192+ for best results.
